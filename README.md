Agile
======

Basic Flow
----------
shift+super+p -> 'Agile: Create New Sprint' -> save sprint
ctrl+super+s -> save story
ctrl+super+o -> select sprint -> select story

Shortcuts
----------
ctrl+cmd+s - save story
crtl+cmd+o - open story
shift+alt+p - all remaining CRUD commands
